Система контроля версий (VCS):

инструмент для отслеживания изменений в файлах проекта, позволяющий вернуться к предыдущим версиям и упрощающий совместную разработку.

Тип VCS Git:

Распределенная VCS (DVCS). Главное отличие от централизованных систем — у каждого разработчика есть полная копия репозитория.

Преимущества Git:

скорость, гибкость ветвления, поддержка распределенной разработки, надежность и широкое распространение.

Отличие Git от GitHub:

Git — это программа (система контроля версий), а GitHub — веб-сервис для размещения Git-репозиториев и совместной работы.
